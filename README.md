## ✨ Features

- **Single-file website** — everything is in `index.html`, no external assets needed
- **Embedded profile photo** — image encoded as base64, no broken image links on deployment
- **Custom animated cursor** — glowing dot + trailing ring that reacts to hover
- **Scroll reveal animations** — sections fade and slide in as you scroll
- **Animated background** — subtle grid lines + floating glow orbs
- **Fully responsive** — works on mobile, tablet, and desktop
- **Smooth navigation** — fixed navbar with anchor scroll links
- **6 complete sections** — Hero, About, Skills, Experience, Projects, Certifications, Contact

## 🚀 How to Deploy (GitHub Pages)

**Step 1 — Create a GitHub repository**
```
Repository name: RajRakshitSahoo.github.io
Visibility: Public
```

**Step 2 — Upload the file**
- Click "uploading an existing file" on the repo page
- Upload `index.html`
- Click **Commit changes**

**Step 3 — Enable GitHub Pages**
- Go to repo **Settings → Pages**
- Branch: `main` → Folder: `/ (root)` → **Save**

**Step 4 — Done! 🎉**
```
Your live link: https://RajRakshitSahoo.github.io
```
> Site goes live within ~1 minute.
